วิธีการติดตั้ง
1.Double click ที่โฟล์เดอร์  src/Habit Task 
2.Double click  ที่โฟล์เดอร์  Habit Task 
3.Double click ที่โฟล์เดอร์  src แล้วกด Download
4.ไฟล์ที่ Download จะอยู่ที่ This PC > Download ซึ่งมาในรูปแบบของไฟล์ .zip
เราต้องแตกไฟล์ก่อนโดยการกด Extract All
5.- เมื่อกด Extract Files เราสามารถเลือกได้เลยว่าจะให้ไฟล์ไปเก็บที่ไหน 
โดยกดที่ Browser… แล้วกด Extract ได้เลย
6.ก็ได้ไฟล์มาเรียบร้อย
7.เปิดโปรแกรมที่จะใช้รันในที่นี้ใช้ Visual Studio Code กดที่  File เลือก Open Folder Habit Task
